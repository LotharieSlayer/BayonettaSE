# Bayonetta PC, PS3 and Wii U Save Editor

Modifies Halo, Difficulty & Items/Consumables.  

### Supported Platforms

PC (in development) inspired and copied from previous [type0dev/Bayo1-editor](https://github.com/type0dev/Bayo1-editor) work.   
PS3 and Wii U maintained by [MarkH221/Bayonetta-Save-Editor](https://github.com/MarkH221/Bayonetta-Save-Editor) forked for C# scripting, interface, and to unify the tool.  

### PR welcome
- Xbox 360 support

### Requirements
.NET 4.8
